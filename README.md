# Library Events Aggregator (Minuteman + BPL)

Streamlit app for day-wise / month-searchable events from Minuteman Library branches and Boston Public Library.

## Run Locally
pip install -r requirements.txt
streamlit run library_events_app.py

## Deploy to Streamlit Community Cloud
1. Push to GitHub.
2. Go to share.streamlit.io → New app → paste repo URL.
